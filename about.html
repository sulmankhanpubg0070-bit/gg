About Family Tree Creator

Ye website aapki family ke haseen tree ko ek intehai simple, elegant aur modern tareeqay se display karti hai. Isko chalane ke liye internet ya database ki koi zaroorat nahi—yeh offline chalne wali ek nayaab website hai.

Outstanding Features

100% Offline Capability

Aapko kisi internet connection, hosting ya backend server ki zaroorat nahi. Browser open karte hi yeh chalne lagti hai.

Smooth Interactive Zooming

Apne multi-generational tree branches ko zoom out, zoom in, standard controls se control karein, aur fit visual zoom ka lutaf uthayein.

Instant High-precision Search

Naam type karte hi matches real-time me highlight ho jate hain aur tree automatic search term par focus hojata hai.

Curated Premium Themes

Umeed se behtar, high-contrast Dark aur light elements, glow grids jo premium responsive screen experience faraham karte hain.

Family Tree Ko Edit Kaise Karein?

1
Open the "app.js" File

Code Editor me project folder ke andar maujood app.js file ko open karein. Yahan poora data standard JS object ki shakal me rakha hua hai.

2
Choose the Head Root Member

Apne tree ke sabse bade bar-urg member (root person) ki ID chunien aur use file ke upar variable ROOT_ID par set karein:

const ROOT_ID = "p1";
3
Write Down Family Members in PEOPLE

PEOPLE database list me apne family members ko is standard elegant format me update karein:

p1: { id: "p1", name: "Muhammad Ali", // Mukammal Name gender: "M", // "M" ya "F" (Male/Female) birth: "1960", // Paidaish ka saal note: "Family Head", // Extra characteristics / ouda partnerId: "p2", // Life partner ki ID children: ["p3", "p4"] // Bachon ki dynamic matching IDs }
4
Add Partner Relations

Releshan ko correct rakhne ke liye visual partner entries dono tarf cross-sync rakhein. Maslan agar p1 ka partner p2 hai, to p2 ka partner entry bhi p1 lazmi hona chahiye, aur dono me children IDs array same define karein.

Built with plain HTML/CSS/JS (no internet needed).
